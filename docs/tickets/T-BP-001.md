# T-BP-001: AppShell layout slot 化(挙動不変リファクタ)

- Status: in-progress
- 担当: Sonnet subagent
- 依存: なし(ADR-0007)

## 背景

Bottom Panel / Status Bar / Header Toolbar を載せるため、KagiApp::render の構造を
「縦 flex のスロット」に再編する。**このチケットは見た目・挙動を一切変えない純リファクタ**
(スロットの器だけ作る。新 UI は後続チケット)。

## スコープ

1. `KagiApp::render` を次の構造に再編(ADR-0007):
   ```
   div(flex_col, size_full)
     .child(render_header_slot(..))    // 今は既存の header bar(repo名等)をそのまま返す
     .child(render_body(..))           // 既存の sidebar|list|panel 一式を関数抽出(中身は移動のみ)
     .child(render_bottom_panel_slot(..)) // 今は空(None を返す)。器だけ
     .child(render_status_bar(..))     // 既存 status footer をそのまま移す(将来の StatusBar の席)
   ```
2. 既存コードの**移動と関数抽出のみ**。ロジック・スタイル・id・イベント配線は変更禁止
3. `render_*_slot` 系の関数シグネチャは後続チケット(T-BP-002/003, T-HT-001)が拡張しやすい形に
   (`&mut KagiApp` 相当 + `&mut Context<KagiApp>` を受ける)
