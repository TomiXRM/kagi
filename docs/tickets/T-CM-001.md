# T-CM-001: Commit row の right-click event 取得

- Status: todo
- 依存: ADR-0020
- 関連: docs/requirements-context-menu.md / ADR-0020〜0026

## スコープ

1. commit row(`render_rows`)に `on_mouse_down(gpui::MouseButton::Right)` を追加
2. クリック座標(window 座標)と row index を `KagiApp.commit_menu: Option<CommitMenuState>` に保存
   (`CommitMenuState { row_index: usize, position: gpui::Point<gpui::Pixels> }` を新設)
3. menu 表示そのものは T-CM-003。本チケットは state 設置 + `[kagi] context-menu: open row=N` ログまで
4. WIP 行・空白部の右クリックは無視

- [ ] `cargo test` 全パス + own-code warning 0(コード変更がある場合)
