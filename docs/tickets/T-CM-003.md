# T-CM-003: Context Menu component(overlay 描画 + dismiss)

- Status: done
- 依存: T-CM-001, T-CM-004, ADR-0020
- 関連: docs/requirements-context-menu.md / ADR-0020〜0026

## スコープ

1. `commit_menu` が Some のとき自前 overlay で menu を描画(モーダル群と同レイヤ、modal より下)
2. ヘッダ行: `<short SHA> <commit title>`(chars 切り詰め)
3. グループ見出し + 区切り線。Dangerous グループは赤系 + ⚠
4. dismiss: 透明 click-away レイヤ / Escape / 項目実行時
5. 画面端 flip(viewport と比較して上/左に反転)
6. disabled 項目: 薄色 + hover tooltip で理由(sidebar の name_tooltip 方式)

## リスク・規約

- Escape は CloseMainDiff と衝突する。menu open 中は menu 優先で処理する順序を明確に
- force push / reset --hard / git clean をコードに追加するのは絶対禁止(ADR-0024 の隔離例外を除き、本チケットでは一切不可)
- 文字列切り詰めは chars() ベース(byte slice は日本語で panic)

## 実装メモ

- `src/ui/context_menu.rs` に自前 overlay renderer を追加し、root render で bottom panel より後、plan modal より前に差し込む。
- click-away と Escape で閉じる。Escape は `CloseMainDiff` handler 内で menu を優先して閉じる。
- ヘッダは `<short SHA> <commit title>` を chars ベースで切り詰め、クリック座標を viewport 内に clamp して表示する。
