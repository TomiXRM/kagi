## リスク・規約

- tooltip は Stateful 要素必須(.id)。menu item に一意 id を振る
- force push / reset --hard / git clean をコードに追加するのは絶対禁止(ADR-0024 の隔離例外を除き、本チケットでは一切不可)
- 文字列切り詰めは chars() ベース(byte slice は日本語で panic)

## 実装メモ

- `ItemState::Disabled(reason)` は `gpui_component::tooltip::Tooltip` で理由を表示する。
- `ItemState::Hidden` は renderer で skip する。Dangerous group と Reset item は警告色・警告記号で表示する。
