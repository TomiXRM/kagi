## リスク・規約

- clipboard は headless で no-op(既知)。ログで代替確認
- force push / reset --hard / git clean をコードに追加するのは絶対禁止(ADR-0024 の隔離例外を除き、本チケットでは一切不可)
- 文字列切り詰めは chars() ベース(byte slice は日本語で panic)

## 実装メモ

- `CommitAction::CopySha` を `KagiApp::dispatch_commit_action` に実装し、menu と Inspector の Copy SHA を同じ dispatch 経由にした。
- 実行時は clipboard に full SHA を書き、`[kagi] copy-sha: <short>` と footer `SHA copied` を出す。
