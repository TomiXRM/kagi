# T-CM-020: Create branch here の plan 統合(checkout オプション込み)

- Status: todo
- 依存: T-CM-003, ADR-0022(既存 plan_create_branch)
- 関連: docs/requirements-context-menu.md / ADR-0020〜0026

## スコープ

1. `CommitAction::CreateBranchHere` → 既存 create-branch dialog を選択 commit を
   start point として開く(既存実装が selected commit 対応済みか確認し、なければ拡張)
2. **T-HT-005 を吸収**: dialog に「作成後 checkout する」checkbox を追加。
   checkout する場合は plan の predicted に反映(branch 作成 + checkout の合成 plan)
3. 実行は既存 plan_create_branch(+ checkout 時は execute 後に plan_checkout 経由の checkout)
