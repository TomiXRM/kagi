## 実装メモ

- `KagiApp::dispatch_commit_action` の `CommitAction::CherryPick` arm を既存 `open_cherry_pick_modal` に接続。
- Inspector の Cherry-pick action を直接 modal open から dispatch 経由に変更し、menu / Inspector の経路を統一。
- 新しい cherry-pick plan は作らず、既存 in-memory plan(T015/T016)を流用。
