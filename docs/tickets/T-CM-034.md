- Status: todo

## スコープ

1. `src/git/ops.rs` に `plan_revert(repo, id)` / `execute_revert(repo, plan, id)`:
   `repo.revert_commit(commit, head, 0, None)` → in-memory index → conflict なら blockers
   (repo 無傷)。クリーンなら tree write → commit「Revert "<summary>"」→
   `checkout_tree(旧 baseline)` → ref 移動(**ref-order 規則厳守**)
2. predicted に「revert commit が新規作成されます」、dirty なら warning
3. merge commit は plan 生成前に menu 側 disabled(T-CM-004)+ plan 側でも blocker(防御の二重化)
4. tests/revert_test.rs(新規): 成功 / conflict Refused / merge blocker / dirty warning / preflight 不一致
5. headless: `KAGI_REVERT=<row>` + AUTO_CONFIRM
