# T-CM-041: Checkout commit(detached)の plan + 実行 + 警告

- Status: todo
- 依存: T-CM-040, ADR-0022
- 関連: docs/requirements-context-menu.md / ADR-0020〜0026

## スコープ

1. `plan_checkout_commit(repo, id)`: predicted「detached HEAD になります」、
   warnings に「branch 作成を推奨(Create branch here)」+ dirty 警告
2. 実行: `checkout_tree(safe)` → `set_head_detached`(ref-order 規則)
3. plan モーダルに「Create branch here…」への導線ボタン(checkout の代替を提示)
4. tests: 成功(detached になる)/ dirty で安全 checkout 失敗 → Failed + repo 無傷 / preflight
5. headless: `KAGI_CHECKOUT_COMMIT=<row>`
