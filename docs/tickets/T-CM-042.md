5. tests/compare_test.rs(新規): files 数 / diff 内容 / repo 無変更

## 実装メモ

- `src/git/diff.rs` に read-only の `compare_commits` / `compare_file_diff` を追加し、`diff_tree_to_tree` で選択 commit → HEAD の changed files / file diff を取得。
- `CompareView` / `CompareTarget` と `MainDiffSource::Compare` を追加。Inspector の Changed Files 領域に `Comparing: <short> ↔ HEAD` と解除ボタンを表示し、ファイルクリックは main diff pane を開く。
- `KAGI_COMPARE_HEAD=<row>` を追加し、fixture で `[kagi] compare: <base> <-> HEAD files=N` を確認。
- `tests/compare_test.rs` で files 数、diff 内容、`git status --porcelain` 不変を確認。
