## リスク・規約

- workdir diff は staged/unstaged の合算であることを表示名で誤解させない(「working tree (staged+unstaged)」表記)
- force push / reset --hard / git clean をコードに追加するのは絶対禁止(ADR-0024 の隔離例外を除き、本チケットでは一切不可)
- 文字列切り詰めは chars() ベース(byte slice は日本語で panic)

## 実装メモ

- `src/git/diff.rs` に `compare_commit_to_workdir` / `compare_commit_to_workdir_file_diff` を追加。`diff_tree_to_workdir_with_index` に `include_untracked` / `recurse_untracked_dirs` / `show_untracked_content` を設定し、staged + unstaged + untracked を含める。
- Compare with working tree は clean repo では `local changes がありません` として開かず、dirty repo だけ `CompareView` を表示。
- `KAGI_COMPARE_WT=<row>` を追加し、dirty fixture で files ログと status 不変、clean fixture で disabled ログを確認。
