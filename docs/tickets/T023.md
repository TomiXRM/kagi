## 実装メモ

### 使用した gpui マウスイベント API

**on_drag** (fluent, `StatefulInteractiveElement` trait):
```rust
fn on_drag<T, W>(
    self,
    value: T,
    constructor: impl Fn(&T, Point<Pixels>, &mut Window, &mut App) -> Entity<W> + 'static,
) -> Self
where
    T: 'static,
    W: 'static + Render,
```
- `value` にドラッグ開始時のカーソルX座標と初期幅を詰めた `DividerDrag` を渡す
- `constructor` はゴーストビュー (`DividerGhost` = 空 div を返す最小 Render 実装) を返す
- `position` 引数がドラッグ開始時のカーソル位置 (`Point<Pixels>`)

**on_drag_move** (fluent, `InteractiveElement` trait):
```rust
fn on_drag_move<T: 'static>(
    self,
    listener: impl Fn(&DragMoveEvent<T>, &mut Window, &mut App) + 'static,
) -> Self
```
- `DragMoveEvent<T>` は `Capture` フェーズで呼ばれるため、要素の外にマウスが出ても追跡可能
- `event.drag(cx)` → `&T` (今回は `&DividerDrag`)
- `event.event.position` → `Point<Pixels>` (ウィンドウ相対カーソル位置)
- `Pixels` の内部フィールドは `pub(crate)` なので `f32::from(pixels)` で変換

**ドラッグ追跡の実装パターン**:
1. ディバイダ div に `.on_drag(DividerDrag { kind, start_x: 0.0, start_width }, ...)` を設置
2. ドラッグ開始コールバック内で `position.x` を `start_x` として格納した `DividerDrag` を構築
3. ルート div に `.on_drag_move::<DividerDrag>(listener)` を設置し、`cursor_x - drag.start_x` でデルタを計算
4. 値が変わったときだけ `cx.notify()` を呼ぶ
